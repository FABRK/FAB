# FAB
ERC-20 Token (FAB)

Install:

    npm install -g truffle@4.1.15
    npm install -g ganache-cli

Steps to run:

    Open terminal , run "ganache-cli -a 1000"
    Open another terminal, cd <fab directory>
    npm install
    truffle compile
    mkdir test-output
    truffle test

All test cases should pass
